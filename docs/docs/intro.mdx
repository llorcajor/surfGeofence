# Introducción a SurfGeofence

¡Bienvenido a la documentación oficial de **SurfGeofence**! 🏄‍♂️📍

**SurfGeofence** es una plataforma IoT (Internet de las Cosas) diseñada para monitorizar y gestionar dispositivos (como rastreadores en tablas de surf) mediante geolocalización. La aplicación permite a los usuarios definir "misiones" o zonas seguras (geofences) y rastrear los dispositivos en tiempo real para garantizar que no salgan de las áreas permitidas.

## ¿Qué encontrarás aquí?

En esta documentación explicaremos de forma sencilla y clara cómo funciona cada parte del programa:

- **Arquitectura:** Cómo se comunican el Frontend, Backend, la Base de Datos y el broker MQTT.
- **Backend:** La API REST, la protección de rutas y la conexión con dispositivos IoT.
- **Frontend:** La interfaz de usuario donde se gestionan los dispositivos y misiones.
- **Despliegue:** Cómo iniciar todo el sistema fácilmente con Docker.
